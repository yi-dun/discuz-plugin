## 易盾反垃圾云服务-discuz插件

#### 安装说明
1. 解压插件包
2. 上传插件文件夹至插件目录 source/plugin
3. 管理中心-->应用-->插件-->找到【易盾反垃圾插件 1.0 】-->安装插件-->启用插件
4. 设置插件，填入从易盾(http://dun.163.com/)申请到的对应配置信息

#### 卸载说明
1. 管理中心-->应用-->插件-->找到【易盾反垃圾插件 1.0 】-->卸载插件
2. 删除对应的插件目录 source/plugin/yidun_dz

#### 兼容说明
1. Discuz! X3.2 测试通过
